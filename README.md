# Computationally Bounded Error Detection and Correction

A survey and presentation exploring modern cryptographic approaches to error detection and correction over computationally bounded adversarial channels.

Prepared for **CS 880** at the **University of Wisconsin–Madison**.

## Overview

Traditional error-correcting codes are fundamentally limited by information-theoretic bounds such as the Singleton bound. Recent research demonstrates that stronger guarantees become possible when adversaries are assumed to be computationally bounded rather than computationally unbounded.

This project surveys two recent papers that construct efficient error-detecting and error-correcting codes under standard cryptographic assumptions, including Learning With Errors (LWE), collision-resistant hashing, and one-way functions.
